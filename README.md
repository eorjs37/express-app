# express.js

## npm init
```bash
npm init
```

## 설치
> 아래 명령어 통해 설치
```bash
npm install express --save
```

## app.js 시작점 만들기

```javascript
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

## express 실행

```bash
node app.js
```

## Database config 구조

```
├─root
│  ├─config
│  │      database.js // 데이터베이스 접속정보 
│  │      dev.js // dev 데이터베이스 정보
│  │      prod.js // prod 데이터베이스 접속 정보
```

```javascript
//database.js
if (process.env.NODE_ENV === 'production') {
  module.exports = require('./prod');
} else {
  module.exports = require('./dev');
}

```

```javascript
//dev.js

  host: 'localhost',
  port: '3306',

  password: '3509',

  connectionLimit: 30,
};

```
